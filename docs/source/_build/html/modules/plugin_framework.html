plugin_framework.plugin module¶

class plugin_framework.plugin.Plugin(spec)[source]¶

Bases: object

Ova klasa predstavlja osnovu za svaki konkretan plugin. Klasa treba da je apstraktna (spram modela) ali ovde smo je realizovali kao konkretnu kako bismo specificirali atribut _spec. U apstraktnim klasama u Pythonu ne postoji inicijalizator pa samim tim, atribute na nivou objekta ne mozemo da definisemo drugacije sem kao metode (property), tako da je ovo realizovano kao regularna klasa.

get_by_symbolic_name(symbolic_name)[source]¶

Vraca plugin koji ima naziv symbolic_name. Ukoliko se podesi da vise pluginova ima isti symbolic_name, vraca se samo prvi.

Returns:Plugin – pronadjeni plugin.
Raises:IndexError – ukoliko ne postoji ni jedan plugin koji je zadovoljio filter.
